# Поездки «Поехали!» — Гексагональная архитектура

**Дисциплина:** Проектирование интернет-систем  
**Лабораторная работа №2**  
**Тема:** Гексагональная архитектура: проектирование портов и адаптеров  
**Вариант:** №35 — «Поездки "Поехали!"»  
**Питч:** Планы, которые сбываются.  
**Ядро домена:** Маршруты, Места, Даты, Бюджет, Заметки

---

## Структура проекта

```
poekhali/
├── domain/                         # Доменный слой (ядро)
│   └── models/
│       ├── trip.py                 # Агрегат: Поездка
│       ├── route.py                # Модель: Маршрут
│       └── note.py                 # Модель: Заметка
│
├── application/                    # Слой приложения
│   ├── port/
│   │   ├── in_/                    # Входящие порты (Use Cases)
│   │   │   ├── create_trip_use_case.py
│   │   │   └── get_trip_use_case.py
│   │   └── out/                    # Исходящие порты (контракты)
│   │       ├── trip_repository.py
│   │       ├── geo_service.py
│   │       └── notification_service.py
│   └── service/
│       └── trip_service.py         # Оркестратор логики приложения
│
├── infrastructure/                 # Инфраструктурный слой
│   └── adapter/
│       ├── in_/                    # Входящие адаптеры
│       │   └── trip_controller.py  # REST-контроллер (симуляция)
│       └── out/                    # Исходящие адаптеры
│           ├── in_memory_trip_repository.py
│           ├── stub_geo_service.py
│           └── email_notification_service.py
│
└── main.py                         # Точка входа и сборка зависимостей
```

---

## Запуск

```bash
python3 main.py
```

Требования: Python 3.8+, сторонние библиотеки не нужны.

---

## Архитектурные принципы

| Слой | Роль |
|---|---|
| **Domain** | Бизнес-сущности (`Trip`, `Route`, `Note`). Не зависит ни от чего. |
| **Application** | Порты (интерфейсы) и сервис, реализующий use cases. Зависит только от домена. |
| **Infrastructure** | Конкретные адаптеры. Зависит от портов приложения. |

Входящие адаптеры → Входящие порты → Сервис → Исходящие порты → Исходящие адаптеры
